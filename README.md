# AvaliacaoPython2
Objetivos

Ao final da atividade, o estudante deverá ser capaz de:
- Compreender o funcionamento da estrutura de dados Deque.
- Implementar uma estrutura linear em Python usando orientação a objetos.
- Realizar inserções e remoções nas duas extremidades da estrutura.
- Tratar situações de erro, como remoção em estrutura vazia.
- Desenvolver um pequeno teste em console para validar o comportamento da classe.
